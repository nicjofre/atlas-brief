# Atlas Brief — Adding a Listing

Two pages matter: **Input** (where you add new listings) and **Database** (the running list of everything you've added).

---

## Adding a brand new listing

On the Input page, three options for where the data comes from:

1. **CoStar Text** — Paste anything you copy from a CoStar property page. Works for both *for-sale* listings (the Property Summary) and *sold comps* (the Sales Comp detail page). Same tab, same paste, the parser figures out which type and pulls everything it can.

2. **CoStar PDF** — Drop in a CoStar Property Summary PDF export. Slightly richer than text since it pulls images and structured tables better.

3. **Broker OM PDF** — Drop in an offering memorandum from a broker (M&M, CBRE, Northmarq, Berkadia, etc.). Pulls financial summary, marketing pitch quotes, broker contacts, unit mix.

After you click **Parse Deal**, it takes 20–60 seconds, then drops you on the listing's detail page.

---

## Adding more data to an existing listing

Open the listing from the Database, click the **Augment** tab. You'll see five paste boxes for CoStar's separate tabs:

- **Contacts tab** — listing/buyer broker, owner, property manager
- **Public Record tab** — full sale history, 5-year tax assessments, owner mailing
- **Loan tab** — debt history with maturity dates
- **Sales Comp page** — sold-deal detail (only for sold properties)
- **Broker OM PDF** — upload an offering memo

You can paste into any combination of these and click **Parse All** — they all run at once. The diff preview shows you exactly what changed. Existing data is preserved unless the new paste explicitly overwrites it.

---

## Photos

On the listing detail, **Photos** tab has 4 slots — one hero (the FIG. 00 image) and 3 secondary photos. Drag any image in, it auto-resizes, you don't need to worry about file size.

For broker headshots: open the listing's **Brokers & Contacts** tab — there's a small circular slot next to each broker. Drop a photo there. The headshot follows that broker across every listing they're on.

---

## Order of operations for a typical deal

1. CoStar Text or PDF gets the property in the system
2. Augment tab fills in the CoStar detail tabs (Contacts, Public Record, Loan)
3. If you have an OM, upload via Augment
4. If sold, paste the Sales Comp page via Augment for buyer/seller/sale notes
5. Drag in a hero photo + broker headshot

Two minutes per listing once you have the sources open.
